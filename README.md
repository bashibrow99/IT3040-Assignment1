# IT3040 - Assignment 1 - Test Automation

This project contains Playwright automation scripts for the Singlish to Sinhala converter validation.

## Prerequisites
- Node.js installed
- VS Code installed

## How to Install
1. Open the terminal in the project folder.
2. Run the following command to install dependencies:
   npm install
   npx playwright install

## How to Run Tests
To run all tests and see the output in the terminal:
npx playwright test --reporter=list
